브랜드 협업 경험

K뷰티 브랜드로부터 협업 제안을 받아, 담당자와 직접 소통하며 결과물을 맞췄습니다.
메디큐브 · 올리브영 세일@o_osol_ 뷰티 채널
실행
채널 인사이트에서 시청 집중 구간(30–40초)을 확인하고, 메디큐브는 상세히·나머지 제품은 간결하게 비중을 차등 구성해 브랜드 가이드라인과 내 채널 톤을 동시에 잡았습니다.
결과
최종 50초로 압축, 담당자에게 "이대로 업로드" 무수정 컨펌. 첫 협업임에도 수정 요청 없이 완수했습니다.
배운점
데이터를 먼저 보고 콘텐츠 구조를 짜면 K뷰티 브랜드의 요구사항과 채널 최적화를 동시에 충족할 수 있다는 것을 확인했습니다.
구아바탭 (영어 학습)@iam_nojob · 브이로그 채널
실행
일상 브이로그 안에 브랜드 소재를 자연스럽게 녹이는 구성으로 1차 제출, 담당자 피드백(제품 등장 장면 강화)을 받아 2차 수정 후 최종 컨펌받았습니다.
결과
가이드라인을 정확히 반영해 게시 완료. 영어 학습 소재라 어학 경험과 콘텐츠 기획이 자연스럽게 연결됐습니다.
배운점
브랜드 의도를 콘텐츠 안에 녹이려면 피드백을 정확히 읽고 빠르게 반영하는 실행력이 핵심임을 배웠습니다.
이삼오구 직무 연결 — 크리에이터·인플루언서 협업을 인플루언서·콘텐츠 제작자 양쪽 관점에서 경험했고, 인플루언서 마케팅·콘텐츠 제작을 실제로 완수했습니다.

뷰티 트렌드 리포트 자동화

매주 K뷰티 트렌드를 자동 수집·분석해 이메일로 받는 시스템을 직접 구축했습니다.
Why 크리에이터의 알고리즘 편향 문제
내 피드에 뜨는 콘텐츠 = 트렌드라고 착각하게 되는 구조. 내 취향이 개입되지 않는 객관적 시장 데이터가 필요했습니다.
How 데이터 소스 선택 + AI 설계
올리브영 베스트→ 소비자가 실제로 돈 쓴 결과
네이버 쇼핑→ 검색·구매 데이터 기반 인기 제품
구글 트렌드→ 관심도 급상승 신호
Gemini AI가 브랜드별 분석 + 콘텐츠 아이디어 자동 생성 → 매주 HTML 이메일 수신
Result 데이터 기반 숏폼 기획
수집 방식매일 수동 피드 확인 → 주 1회 자동 리포트 수신
기획 방식구글 트렌드 급상승 브랜드 + AI 콘텐츠 아이디어 → 숏폼 소재 선정
커버리지내 알고리즘 밖 브랜드까지 놓치지 않고 자동 포착
실제 수신 리포트 — 2026.06.17
이삼오구 직무 연결 — 뷰티 트렌드를 데이터로 읽고, AI를 업무에 맞게 직접 구축해 매주 활용하고 있습니다.

오프라인 매장 운영

스탬프 마케팅 기획 (크림필터) 2024~ 마케팅 기획
기획

실행
